# SISTEMA DE LOGIN





## Tecnologias Usadas

- React JS
- Node.js
- Routes
- Styled components

## Instalando projeto

### Usando yarn

- Rode `yarn install` para instalar todas as dependencias

### Usando npm

- Rode `npm install` para instalar todas as dependencias

## Rodando projeto

### Instalando as dependencias

- Rode `yarn start` ou `npm start` para inicializar o servidor
